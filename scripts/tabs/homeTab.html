HAR日志文件样例

  • Inline scripts block - 内联脚本阻止页面加载。
  • Blocking time - 最大并联连接数限制的影响。
  • Browser cache - 浏览器缓存对页面加载的影响（同一页面加载三次）。
  • Single page - 单页加载（空缓存）。
